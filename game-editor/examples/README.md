# 示例案例目录

此目录用于存放游戏编辑器的示例案例，每个案例都包含完整的节点图配置和相关资源。

## 案例结构说明

每个案例文件夹包含：
- `graph.json` - 节点图配置文件
- `description.md` - 案例说明文档
- `assets/` - 案例相关的资源文件（图片、音频等）
- `preview.png` - 案例预览图（可选）
